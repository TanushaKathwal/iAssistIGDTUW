Overview
iAssistIGDTUW is a student utility Android app built exclusively for IGDTUW students, designed to make campus life easier by bringing all essential tools and information into one place.

Features
- Event Finder — stay updated on all college events
- SGPA Calculator — calculate semester grades instantly
- Lost & Found — report and find lost items on campus
- Room Finder — locate rooms across campus easily
- Notifications — real-time college updates and alerts
- Latest Pics — fetches latest posts directly from IGDTUW Instagram
- Profile — personalized student profile
- Goal Tracker — set and track academic goals
- Academic Corner — study resources and academic info
- Circulars — access official college circulars
- Life in Motion — students share and discover nearby places to visit

Tech Stack
- **Language:** Kotlin
- **IDE:** Android Studio
- **Database:** Firebase
- **Platform:** Android
